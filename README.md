# Counter-Up-Timer
Count Up Timer - Using Vanilla JS ( Mostly ) and Jquery 
<hr>
Features :
<ul type="square">
   <li>Counter stay on even URL changes  </li>
   <li>Have Ajax method pre-made for storing time data </li>

   <li>Notification animation added here</li>
</ul>
